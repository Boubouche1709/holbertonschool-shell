Introduction
This guide is designed to help you master the basics of navigating a Unix system, manipulating files, using text editors like Emacs and vi, and managing source code with Git and GitHub. Through this document, you will learn how to use the terminal and various essential tools for development on Linux.

Linux File Navigation and Manipulation
Navigation Commands
pwd: Displays the absolute path of the current working directory.
ls: Lists the files and directories in the current directory.
cd: Changes the working directory.
less: Displays the contents of a file page by page.
touch: Creates an empty file.
cp: Copies files or directories.
mv: Moves or renames files or directories.
rm: Removes files or directories.
mkdir: Creates a directory.
rmdir: Removes an empty directory.
Recommended resources to learn these commands are the manual pages (e.g., man ls, man cd).
